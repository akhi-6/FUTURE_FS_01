# 🌐 Personal Portfolio Website

## 📌 Internship Task

This project was developed as part of **Task 1 – Full Stack Web Development Internship** at **Future Interns**.

The objective of this task was to build a **professional portfolio website** that showcases my skills, projects, and provides a way for recruiters and clients to connect with me.

---

## 🚀 About the Project

This is a **personal portfolio website** designed to represent my professional profile online.

It highlights:

* My background and skills
* Real-world projects I have built
* Contact information for collaboration or opportunities

The website is built with a focus on **clean UI, modern design, and user-friendly experience**.

---

## ✨ Features

✔ **Professional Homepage**

* Introduction with profile image
* Clean and modern hero section

✔ **About Section**

* Overview of my background
* Skills and technologies

✔ **Projects Showcase**

* Real-world projects including:

  * PyFixAI (Python Auto Debugging Library)
  * Face Recognition & Hand Gesture System
  * AR Hand Tracking Project
  * Avanthi Chatbot (College Assistant)
  * Student Record CRUD Application

✔ **Skills Section**

* Technical skills and tools

✔ **Contact Section (No Form)**

* Direct contact details including:

  * Email
  * LinkedIn
  * GitHub
  * Mobile number

✔ **Resume Integration**

* View/download resume directly from the website

✔ **Modern UI/UX**

* Matte dark green + black theme
* Glow effects and clean layout
* Responsive design

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** – Basic interactivity
* **Formspree (initially)** – For contact form (later replaced with direct contact info)
